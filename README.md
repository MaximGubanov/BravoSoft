# BravoSoft

## Описание
***BACKEND***: 


    Express.js - для реализации API
    Prisma ORM - для работы с БД
    Postgres - СУБД


***FRONTEND***:


    React.js, Redux


## Как запустить?

    Просто перейдите в директорию BravoSoft и выполните:

        $ cd BravoSoft
        $ docker-compose up --build

    Затем перейдите в браузер по ссылке http://localhost:8888

        Имя БД - postgres
        Пользователь - user
        Пароль - user
        Порт - 5432
